## ✨ VERSÃO MELHORADA - COMPLETA

```
Desenvolvedor de Sistemas | iTracker - Grupo ICTSI
jan 2024 - Presente (1 ano 3 meses) | Rio de Janeiro, Brasil

Desenvolvimento de soluções tecnológicas para o maior terminal portuário privado do Brasil (ICTSI), otimizando operações que movimentam milhares de contêineres diariamente. Foco em automação, monitoramento em tempo real e sistemas enterprise escaláveis.

🎯 IMPACTO & RESULTADOS:

• Desenvolvi 8+ sistemas enterprise que atendem 200+ usuários diários, reduzindo processos manuais em 80% e economizando 40+ horas semanais de trabalho operacional

• Criei automações RPA que processam 500+ consultas diárias de forma autônoma, eliminando tarefas repetitivas e reduzindo erros humanos em 95%

• Implementei dashboards em tempo real que fornecem visibilidade 360° das operações, permitindo tomada de decisão baseada em dados para gestão executiva

• Reduzi custos operacionais em 70% ao desenvolver soluções internas que substituíram sistemas terceirizados caros

💻 PRINCIPAIS PROJETOS DESENVOLVIDOS:

→ Digital Signage Platform (TV Corporativa)
Plataforma de sinalização digital multi-empresa com gestão centralizada de conteúdo para 15+ telas corporativas
Stack: Python, Flask, React, Socket.IO, Redis, FFmpeg, Chromecast
Impacto: Comunicação corporativa em tempo real para 300+ colaboradores

→ Torre de Controle Operacional
Dashboard executivo para monitoramento em tempo real de 20+ aplicações críticas e rotinas automatizadas
Stack: Python, Streamlit, Plotly, Selenium, Google APIs, Prometheus
Impacto: Redução de 60% no tempo de identificação de falhas operacionais

→ CargoSnap Integration
Sistema de gestão de inspeções de contêineres com integração API externa e workflow de aprovação
Stack: Django REST Framework, PostgreSQL, JWT, API Integration
Impacto: Processamento de 100+ inspeções diárias com rastreabilidade completa

→ Bot Consulta Multas DETRAN (RPA)
Automação RPA para consultas simultâneas em portais (RJ, SP, ES) com geração automática de relatórios Excel
Stack: Python, Selenium, Pandas, OpenPyXL, Multi-threading
Impacto: Economia de 15 horas semanais, processando 500+ consultas/dia

→ GPS Tracking System
Rastreamento de frotas em tempo real (50+ veículos) com detecção automática de eventos e alertas
Stack: Python, Flask, API Telematics, Geolocation, WebSocket
Impacto: Visibilidade 100% da frota com alertas proativos de desvios

→ GateLog - Controle de Acesso Portuário
Sistema de controle de entrada/saída com autenticação multi-empresa e logs auditáveis
Stack: Vue.js 3, Vite, TailwindCSS, Django REST Framework, JWT, PostgreSQL
Impacto: Gestão de 200+ acessos diários com segurança e rastreabilidade

→ Sistema de Não-Conformidades
Gestão de qualidade operacional com workflows de aprovação multinível e dashboards gerenciais
Stack: Django, PostgreSQL, Chart.js, Workflow Engine
Impacto: Redução de 50% no tempo de resolução de não-conformidades

→ Sistema de Atendimento/GR
Controle de acesso por perfis com gestão multi-empresa e permissões granulares
Stack: Flask, MySQL, JWT, RBAC (Role-Based Access Control)
Impacto: Gestão segura de 100+ usuários com diferentes níveis de acesso

🔧 RESPONSABILIDADES TÉCNICAS:

• Arquitetura Full-Stack: Design e implementação de soluções completas do banco de dados ao frontend, seguindo princípios SOLID e Clean Architecture

• DevOps & Infraestrutura: Deploy e manutenção de aplicações em servidores Linux (Ubuntu) e Windows Server com Nginx, garantindo alta disponibilidade

• APIs & Integrações: Desenvolvimento de APIs RESTful robustas e integrações com sistemas externos (ERPs, APIs governamentais, serviços de terceiros)

• Automação Inteligente: Criação de RPAs com Selenium e PyAutoGUI para automatizar processos bancários, consultas governamentais e tarefas repetitivas

• Monitoramento & Observabilidade: Implementação de dashboards com Prometheus, Grafana e Streamlit para visibilidade operacional em tempo real

🛠️ STACK TECNOLÓGICO:

Backend: Python (Django, Flask, FastAPI), Node.js
Frontend: React, Vue.js 3, Streamlit, Material UI, TailwindCSS
Database: PostgreSQL, MySQL, Redis
DevOps: Docker, Linux (Ubuntu), Windows Server, Nginx
Automação: Selenium, PyAutoGUI, Celery, APScheduler
Integrações: APIs REST, WebSocket, JWT, OAuth
Monitoramento: Prometheus, Grafana, Plotly
Outros: FFmpeg, Chromecast API, Google APIs, Telematics API

💡 COMPETÊNCIAS DESENVOLVIDAS:

• Visão de Negócio: Transformar requisitos de negócio em soluções técnicas escaláveis alinhadas aos objetivos estratégicos da empresa

• Resolução de Problemas: Identificar gargalos operacionais e propor soluções inovadoras que geram valor mensurável

• Aprendizado Rápido: Domínio de novas tecnologias e frameworks conforme demanda dos projetos (Go, Java, Flutter)

• Trabalho Autônomo: Gestão completa do ciclo de vida de projetos, desde levantamento de requisitos até deploy em produção

• Comunicação Técnica: Documentação clara de sistemas e apresentação de soluções para stakeholders técnicos e não-técnicos
```
