# Project Ranking — Leonardo Fragoso Portfolio

**Date:** 2026-06-25  
**Goal:** Identify which projects best demonstrate Senior Backend Engineer capabilities.

## Ranking Criteria

Each project was evaluated on a 1-5 scale across:

- **Technical Complexity:** Architecture depth, number of services, integrations.
- **Real-World Impact:** Production usage, business value, user scale.
- **AI/ML Usage:** Use of AI, automation, or data-driven features.
- **Scalability:** Multi-tenant design, caching, async processing, deployment.
- **Backend Focus:** How much the project emphasizes backend engineering.
- **Code Quality:** Evidence of testing, documentation, patterns.

## Recommendation for Portfolio Restructure

### Keep Prominent (Hero + Featured)

1. **ProFlow** — production SaaS, strongest business case.
2. **AgentesIA / Oráculo IA** — AI authority, cutting-edge backend.
3. **LogiFlow CRM** — enterprise system design, multi-app ecosystem.

### Keep as Supporting Projects

- Digital Signage Platform
- APM Platform
- BI-as-a-Service
- Assistente Financeiro WhatsApp
- Não-Conformidades

### Consider Deprioritizing or Removing

- Dashboard Multas
- GNLink
- PyScript.tech
- Base Corporativa
- TaskManager (Go)
- Andaimes Pini

### Rationale

A senior backend portfolio should look like a curated exhibition, not a catalog of 20 projects. The top 3 projects already cover:
- SaaS + payments + AI (ProFlow)
- AI agents + FastAPI (AgentesIA)
- Enterprise logistics + multi-app (LogiFlow)

Everything else should support these narratives or be removed.
